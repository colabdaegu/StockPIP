# StockPIPApp
![Preview](src/main/resources/logo/Stock_Logo_fill.ico)

## 📱 PIP 애플리케이션 구현 상세 기획서
프로젝트명: **Stock PIP**  
개발 환경: **Java 17 이상, JavaFX, JFoenix, IntelliJ IDEA, GSON, Finnhub REST API, Gemini API**

---

## ✅ 프로젝트 개요
사용자가 감시하고 싶은 종목(주식, 코인 등)의 가격 정보를 실시간으로 받아와  
항상 화면 위에 떠 있는 초경량 **PIP(Picture-in-Picture)** 형태로 표시하는 데스크탑 애플리케이션을 개발한다.

투명한 배경에 텍스트만 떠 있는 형태로 정보를 제공하여  
다른 작업을 방해하지 않고도 직관적으로 가격을 확인할 수 있도록 한다.

세밀한 JSON 처리를 통해 프로그램을 종료해도  
나중에 다시 실행했을 때 추가한 종목 및 환경설정이 그대로 유지되도록 구현한다.

추가로, 지도 학습 AI를 연동하여 단기 전망치를 예측해볼 수 있다.

---

## ✅ 주요 기능 요약
- 메인 화면: 종목 추가, 알림 주기 입력, 저장/초기화 기능, AI 분석
- PIP 오버레이 창: 투명 배경, 실시간 가격 표시, 상승/하락/변동 시각 효과
- 팝업 창: 목표가 알림, 손절가 알림
- 종목 가격 연동: 외부 API(Finnhub 등) 사용
- 조건 기반 알림: 목표가 도달 시 색상/애니메이션 효과 제공
- 설정 저장: JSON 또는 Preferences API 저장/불러오기
- 설정 팝업: 종목/조건 수정, 새로고침 주기, 알림 방식, 다양한 PIP 설정
- 위치/크기 조절: 마우스로 PIP 창 이동 및 크기 조절 가능
- AI 분석: 저장된 종목 학습 후 전망치 예측

## ✅ 화면 구성

### 1. 메인 화면
- 감시할 종목명 입력 및 선택
- 목표가 / 손절가 등 조건 다중 입력
- 최근 입력 종목 자동 저장
- 저장 / 초기화 버튼 제공
- PIP 사용자 맞춤형 환경설정 제공
- 상세 시세 정보 및 로그 제공
- AI 분석 결과 제공

---

### 2️. PIP 오버레이 창 (항상 위 고정)
- 현재 가격 실시간 표시
- 가격 변동 시 시각 효과
   - 상승: 🔴 빨간색
   - 하락: 🔵 파란색
   - 변동: 텍스트 깜빡임
- 기본 반투명 / 마우스 오버 시 불투명
- 위치/크기 자유 조절

---

### 3. 팝업 창
- 목표가 / 손절가 알림 제공
- Alert / SystemTray 방식 선택 가능
- 환경설정에서 알림 방식 변경 가능
- 알림 일시 중지 기능
- 알림 후 종목 자동 제거 옵션
- 알림 완전 비활성화 가능

## ✅ 개발 일정
- 1주차: 기획, 개발환경 세팅, UI 기본 구조

## ✅ 사용 기술
- 언어: Java 21
- GUI: JavaFX 21
- API: REST (Finnhub), Gemini AI
- JSON: Gson
- 저장 방식: JSON 파일 / Preferences API
- 배포: `.jar`, `.exe` (Launch4j, jpackage)
